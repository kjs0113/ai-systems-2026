## 실습과제 안내

총 12개의 실습과제를 통해 강의 이론을 직접 구현한다. 모든 과제는 **GitHub PR**로 제출한다.

## 제출 방법

```bash
# 1. 저장소 Fork 후 클론
git clone https://github.com/[YOUR_USERNAME]/ai-systems-2026.git

# 2. 과제 폴더에 작업
mkdir -p assignments/lab-01/[학번]
cd assignments/lab-01/[학번]

# 3. 코드 작성 후 커밋
git add .
git commit -m "[제출] Lab 01 - 20230001 홍길동"

# 4. PR 생성
git push origin main
# GitHub에서 PR 생성
```

## 평가 기준

- **기능 구현** (60%): 요구사항 충족
- **코드 품질** (20%): 가독성, 주석, 타입 힌트
- **문서화** (20%): README, 설계 결정 사항 기록

## 제출 마감 정책

- 마감일 기준 **3일 이내** 제출 시 10% 감점
- **7일 초과** 시 50% 감점
- **2주 초과** 시 0점 처리
