## 실습 단계

<Steps>
1. **Node.js 설치 확인**

   ```bash
   node --version  # v20.x.x 이상
   pnpm --version  # 10.x.x 이상
   ```

   Node.js가 없다면 [nodejs.org](https://nodejs.org)에서 설치.

2. **AI 코딩 CLI 도구 설치**

   <Tabs>
     <TabItem label="Claude Code (권장)">
       ```bash
       pnpm add -g @anthropic-ai/claude-code
       claude --version
       ```
     </TabItem>
     <TabItem label="Gemini CLI">
       ```bash
       pnpm add -g @google/gemini-cli
       gemini --version
       ```
     </TabItem>
     <TabItem label="Codex CLI">
       ```bash
       pnpm add -g @openai/codex
       codex --version
       ```
     </TabItem>
     <TabItem label="OpenCode">
       ```bash
       brew install opencode
       ```
     </TabItem>
   </Tabs>

   <Aside type="note">도구별 상세 비교는 [AI 코딩 도구 선택 가이드](/reference/tool-selection) 참조.</Aside>

3. **API 키 설정**

   <Tabs>
     <TabItem label="Claude Code">
       [console.anthropic.com](https://console.anthropic.com)에서 API 키 발급 후:
       ```bash
       # ~/.bashrc 또는 ~/.zshrc에 추가
       export ANTHROPIC_API_KEY="sk-ant-..."
       source ~/.bashrc
       ```
     </TabItem>
     <TabItem label="Gemini CLI">
       [aistudio.google.com](https://aistudio.google.com)에서 API 키 발급 후:
       ```bash
       export GEMINI_API_KEY="..."
       # 또는 첫 실행 시 브라우저 OAuth 인증
       ```
     </TabItem>
     <TabItem label="Codex CLI">
       [platform.openai.com](https://platform.openai.com)에서 API 키 발급 후:
       ```bash
       export OPENAI_API_KEY="sk-..."
       ```
     </TabItem>
     <TabItem label="OpenCode">
       ```bash
       # 사용할 모델 제공자에 따라 설정
       export OPENAI_API_KEY="sk-..."  # OpenAI 모델 사용 시
       ```
     </TabItem>
   </Tabs>

4. **첫 실행 테스트**

   <Tabs>
     <TabItem label="Claude Code">
       ```bash
       mkdir ~/hello-agent && cd ~/hello-agent
       claude "Python으로 'Hello, AI Engineering 2026!' 출력하는 파일을 만들어줘."
       python hello.py
       ```
     </TabItem>
     <TabItem label="Gemini CLI">
       ```bash
       mkdir ~/hello-agent && cd ~/hello-agent
       gemini
       # 대화형 모드에서: "Python으로 'Hello, AI Engineering 2026!' 출력하는 파일을 만들어줘."
       ```
     </TabItem>
     <TabItem label="Codex CLI">
       ```bash
       mkdir ~/hello-agent && cd ~/hello-agent
       codex "Python으로 'Hello, AI Engineering 2026!' 출력하는 파일을 만들어줘."
       python hello.py
       ```
     </TabItem>
     <TabItem label="OpenCode">
       ```bash
       mkdir ~/hello-agent && cd ~/hello-agent
       opencode
       # TUI에서: "Python으로 'Hello, AI Engineering 2026!' 출력하는 파일을 만들어줘."
       ```
     </TabItem>
   </Tabs>

5. **관찰 과제: AI의 자율적 결정 기록**

   AI가 생성한 코드를 열어서 다음을 관찰하라:
   - AI가 **스스로 결정**한 것은 무엇인가? (파일명, 함수 구조, 변수명, 주석 언어 등)
   - 사용자가 **명시적으로 지시**한 것과 AI가 **자율적으로 결정**한 것을 구분하여 기록
   - 이 관찰이 1주차 이론의 "AI 모델 vs AI 시스템" 구분과 어떻게 연결되는가?

   ```markdown
   # 관찰 기록 (observations.md에 작성)
   ## 내가 지시한 것
   - "Hello, AI Engineering 2026! 출력하는 파일"

   ## AI가 자율적으로 결정한 것
   - 파일명: ???
   - 변수명/함수명: ???
   - 코드 구조: ???
   - 주석 언어: ???
   - 기타: ???
   ```

6. **DGX 서버 접속**

   <Tabs>
     <TabItem label="SSH">
       ```bash
       ssh [학번]@dgx.chu.ac.kr
       # 초기 비밀번호: 학번
       passwd  # 즉시 변경 필수!
       ```
     </TabItem>
     <TabItem label="SSH Config">
       ```
       # ~/.ssh/config
       Host dgx
         HostName dgx.chu.ac.kr
         User [학번]
         IdentityFile ~/.ssh/id_ed25519
       ```
     </TabItem>
   </Tabs>

7. **저장소 Fork 및 클론**

   ```bash
   # GitHub에서 Fork 후
   git clone https://github.com/[YOUR_USERNAME]/ai-systems-2026.git
   cd ai-systems-2026
   pnpm install
   pnpm run dev  # 로컬 사이트 확인
   ```
</Steps>
