## 공통 피드백

<Aside type="caution" title="제출 시 주의사항">
1. **최신 main 기준으로 작업** — PR 생성 전 반드시 `git pull origin main`. 오래된 base의 PR은 다른 학생 제출물을 대량 삭제하는 대재앙을 유발합니다
2. **자기 학번 폴더만 변경** — fork에서 작업 시 다른 학생 파일이나 강의자료 수정 금지
3. **`.pyc` 파일 제외** — `.gitignore`에 `__pycache__/`와 `*.pyc` 추가
4. **한글 파일명 금지** — macOS NFD 분해 이슈. 영문 파일명 사용
5. **`src/content/docs/`에 과제 파일 넣지 않기** — 강의 사이트 소스 디렉토리
6. **PR 전 확인** — `git diff --stat origin/main`으로 변경 파일 목록 검토
</Aside>
