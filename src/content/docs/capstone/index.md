## Ralphthon 캡스톤 프로젝트

**Ralphthon**은 16주 강의의 최종 캡스톤 프로젝트다. Ralph 루프 방법론을 기반으로 **실제 소프트웨어 문제를 해결하는 자율 에이전틱 시스템**을 팀으로 설계하고 구현한다.

## 일정

| 주차 | 내용 | 마감 |
|------|------|------|
| 13주차 | 팀 구성 + 아키텍처 설계 문서 제출 | 2026-06-03 |
| 14주차 | Ralphthon 실행 + 중간 보고 | 2026-06-10 |
| 15주차 | 통합 테스트 + 발표 자료 제출 | 2026-06-17 |
| 16주차 | 최종 발표 + 동료 평가 | 2026-06-24 |

## 평가 기준

캡스톤은 총 30점 (전체 성적의 30%):

| 항목 | 배점 | 설명 |
|------|------|------|
| 기술적 완성도 | 12점 | 에이전트 파이프라인 실제 동작 |
| 문제 적합성 | 6점 | 주제 선정의 적절성과 해결 효과 |
| 발표 품질 | 6점 | 명확한 설명, 라이브 데모 |
| 동료 평가 | 6점 | 팀원 기여도 평가 |
