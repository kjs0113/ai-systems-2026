## Common Feedback

<Aside type="caution" title="Submission Guidelines">
1. **Work from latest main** — Always `git pull origin main` before creating a PR. PRs from outdated bases cause catastrophic mass deletion of other students' submissions
2. **Only modify your own student ID folder** — Don't modify other students' files or course materials when working from a fork
3. **Exclude `.pyc` files** — Add `__pycache__/` and `*.pyc` to `.gitignore`
4. **No Korean filenames** — macOS NFD decomposition issues. Use English filenames
5. **Don't put assignment files in `src/content/docs/`** — This is the course site source directory
6. **Check before PR** — Review changed file list with `git diff --stat origin/main`
</Aside>
