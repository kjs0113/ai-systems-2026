# Ralph Loop Tasks

이 파일은 Ralph 루프의 에이전트가 수행해야 할 태스크 명세를 포함합니다.  각 태스크는 명확한 사양과 수락 조건을 가지고 있으며, 루프는 모든 조건을 만족할 때까지 반복합니다.

## 태스크 1: `fibonacci(n)` 구현

### 사양

Python 함수 `fibonacci(n)`을 작성하여 n번째 피보나치 수를 반환하십시오.

- 입력은 `n >= 0`인 정수 하나입니다.
- 효율적인 알고리즘을 사용하십시오(메모이제이션이나 반복문을 권장하며, 불필요하게 많은 재귀 호출을 피하십시오).
- `n`이 음수이거나 정수가 아닌 경우 예외를 발생시켜야 합니다.
- 함수는 동작을 설명하는 간단한 docstring을 포함해야 합니다.

### 수락 조건

- `fibonacci(0)` 은 `0`, `fibonacci(1)` 은 `1`을 반환합니다.
- `fibonacci(10)` 의 결과는 `55`입니다.
- `fibonacci(30)`과 같이 값이 큰 입력도 눈에 띄는 지연 없이 처리할 수 있습니다.
- 제공된 모든 단위 테스트를 통과해야 합니다.

## 태스크 2: `is_prime(n)` 구현

### 사양

정수가 소수인지 판별하는 Python 함수 `is_prime(n)`을 작성하십시오.

- 입력 `n`은 0 이상의 정수입니다.
- `n <= 1`일 때는 항상 `False`를 반환합니다.
- 소수 판별은 `sqrt(n)` 까지 나누어 보는 시도(div trial) 알고리즘을 사용합니다. `n`이 10,000 이하의 값에 대해서 효율적으로 동작해야 합니다.
- 입력이 정수가 아니거나 음수인 경우 적절한 예외를 발생시킵니다.
- 함수에는 역할을 설명하는 docstring을 포함해야 합니다.

### 수락 조건

- `is_prime(2)` 는 `True`를 반환합니다.
- `is_prime(15)` 는 `False`를 반환합니다.
- `is_prime(9973)` 는 `True`를 반환합니다.
- 모든 테스트를 통과해야 하며, 부적절한 입력에 대해 예외를 올바르게 발생시켜야 합니다.

## 태스크 3: `bubble_sort(arr)` 구현

### 사양

버블 정렬을 이용해 숫자 리스트를 정렬하는 Python 함수 `bubble_sort(arr)`를 작성하십시오.

- 입력은 숫자(정수 또는 실수)로 구성된 리스트입니다.
- 함수는 리스트를 오름차순으로 정렬해야 합니다. 원본 리스트를 수정하거나 정렬된 새 리스트를 반환할 수 있으나, 어느 방식인지는 docstring에 명시해야 합니다.
- 버블 정렬 알고리즘을 사용해야 하며, 파이썬 내장 `sorted()` 함수를 사용해서는 안 됩니다.
- 함수에는 목적과 동작을 설명하는 docstring을 포함해야 합니다.

### 수락 조건

- 빈 리스트나 요소가 하나인 리스트에 대해서도 올바르게 동작해야 합니다.
- 정렬 결과는 입력 리스트의 길이를 보존하며, 값이 변형되어서는 안 됩니다.
- `[3, 1, 2]`를 입력으로 주었을 때 `[1, 2, 3]`을 반환하거나 이에 상응하는 결과를 돌려야 합니다.
- 제공된 테스트를 모두 통과해야 합니다.
