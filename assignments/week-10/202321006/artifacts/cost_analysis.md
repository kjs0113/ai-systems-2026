# 비용 분석 보고서 (1년 기준)

### 1. 상용 API 비용 (Claude-3.5-Sonnet)
- **전제:** 하루 1,000만 토큰 사용 (Input 70%, Output 30%)
- **단가:** Input $3/1M, Output $15/1M
- **계산:** ($3 \times 7) + ($15 \times 3) = $66/일
- **연간 총액:** $66 \times 365 = **$24,090 (약 3,300만원)**

### 2. DGX 운영 비용 (로컬 H100 1개 할당 기준)
- **전제:** 기보유 DGX H100 서버 활용 (전기료 및 유지보수비만 계산)
- **전기료:** H100 (Max 700W) + 서버 본체 = 약 1kW 소비
- **시간당 비용:** 1kWh당 200원 가정 시 시간당 200원
- **연간 전기료:** 200원 \times 24시간 \times 365일 = **약 175만원**
- **유지보수비:** 하드웨어 상각 및 인건비 포함 시 약 **연간 500만원**
- **연간 총액:** **약 675만원**

### 3. 결론
- 로컬 모델 사용 시 상용 API 대비 **약 80% 이상의 비용 절감**이 가능함.
- 초기 구축 비용(서버 구매)을 제외하면, 사용량이 많을수록 오픈소스 모델이 압도적으로 유리함.
